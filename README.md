# 🎯 Striver Pattern Programs in Java

A collection of **22 Pattern Printing Programs** implemented in **Java**, inspired by the **Striver DSA Sheet**. This repository is designed to help beginners strengthen their understanding of nested loops, pattern logic, and problem-solving.

---

## 📌 Features

- ✅ 22 different pattern printing programs
- ✅ Clean and easy-to-understand Java code
- ✅ Beginner-friendly implementation
- ✅ Covers stars, numbers, alphabets, pyramids, diamonds, butterflies, and concentric patterns
- ✅ Perfect for DSA and coding interview preparation

---

## 🛠️ Technologies Used

- Java
- VS Code / IntelliJ IDEA / Eclipse
- JDK 8+
